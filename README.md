# 海兰德反馈工单
## 提交流程
如果已有的Issue无法描述你的问题</br>
Issue页面-New Issue 新建工单</br>
Assignees 选择负责你这部分的开发者 例如是移动端就可以assign "silverwolf233“</br>
详细描述问题</br>
选择适合你问题的Label</br>
## 工单模板
**描述问题**
**如何复现**
**期待效果**
## 申请参与
新建工单-签署DCO(Developer Certificate Of Origin)</br>
Assignees 你希望参加的开发的部分负责的开发者</br>
我们会在1到2周内给你答复</br>
DCO全文:请看DCO.md</br>
